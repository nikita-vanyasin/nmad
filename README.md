# nmad Telegram bot

Add [bot](https://t.me/nmad_o_bot) to your chat to track nomad locations!

[http://nmad.link/](http://nmad.link/)

Available commands:
```
/set_city Budva
/get_city Tbilisi
/get_country Turkey
/list
/map
```
